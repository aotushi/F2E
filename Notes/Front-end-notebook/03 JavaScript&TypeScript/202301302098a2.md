#### 类型错误
一般有两种情况:
* 操作符使用在了不适当的类型变量上,例如对数字类型使用concat操作
* 操作的变量遇到不可预期的null或undefind值

例如:
```js
const obj = {};
obj.concat([1]); // Uncaught TypeError: obj.concat is not a function

const a = null;
a.nickname; // Uncaught TypeError: Cannot read property 'nickname' of null
```
