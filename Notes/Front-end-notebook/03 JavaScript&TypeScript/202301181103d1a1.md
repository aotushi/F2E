### Symbol.iterator


`Symbol.iterator`静态数据属性表示指定==返回对象迭代器的方法==的已知符号. ??
如果这个属性被设置在一个对象上, 那么它是可迭代的并且能用于for...of循环和其它语法.


#### 数据属性
| 数据属性     | 值           |
| ------------ | ------------ |
| value        | `@@iterator` |
| writable     | false        |
| enumerable   | false        |
| configurable | false        |
