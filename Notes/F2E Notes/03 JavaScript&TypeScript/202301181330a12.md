### 修改类constructor函数返回值

可以通过在constructor函数中通过return返回一个新对象来覆盖默认的对象
```js
class Person {
	constructor(name) {
		this.name = name
		this.sex  = 'man'
	}
}
```

使用return返回一个对象
```js
class Person {
	constructor(name) {
		return {
			name,
			sex: 'man'
		}
	}
}
```
